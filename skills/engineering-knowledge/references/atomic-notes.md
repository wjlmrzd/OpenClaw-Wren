# 原子笔记模板

## 完整模板

```markdown
---
title: "[条款标题]"
source: "[原始文档规范名称]"
version: "[版本编号]"
section: "[条款编号]"
scope: "[适用范围]"
category: [专业分类]
docId: [文档唯一ID-条款号]
status: active
tags: [原子, 文档类型, 专业分类]
created: YYYY-MM-DD
related:
  - [[文档ID_上级条款]]
  - [[文档ID_下级条款]]
  - [[知识/相关概念]]
---

# [条款标题]

## 原子内容

[条款具体内容，包括数值、要求、条件等]

## 要点总结

- [ ] 要点 1
- [ ] 要点 2
- [ ] 要点 3

## 关联条款

| 条款 | 关系 | 说明 |
|------|------|------|

| [[文档ID_条款号]] | 下级 | 子条款 |

## 来源追溯

- **文档**: [[文档库/文档名]]
- **原文条款**: [条款号]
- **版本**: [版本]
- **适用范围**: [范围]

## 补充说明

[其他相关信息]
```

## 文档编号识别规则

| 格式 | 示例 | 说明 |
|------|------|------|
| GB/T XXXXX-YYYY | GB/T 50001-2017 | 国家标准 |
| JTG XXXX | JTG D60-2015 | 交通行业标准 |
| JG XXXX | JGJ 130-2011 | 建筑行业标准 |
| QB XXXX | QB/T 2577-2012 | 轻工行业标准 |
| DB XX/XX | DB11/1027-2013 | 地方标准 |

## 版本提取正则

```
v?(\d+\.)?\d+[a-z]?|Rev\.?\s*\d+|第\s*\d+\s*版
```

## 条款编号提取正则

```
\d+(\.\d+)+ 或 [一二三四五六七八九十]+、
```

## 原子笔记命名规则

```
{文档编号}_{条款编号}_{主题词}.md

示例:
GB50001_5.2.3_混凝土强度等级.md
JTG_B01_3.1.2_设计速度标准.md
```

## 关联分析规则

| 条件 | 关联类型 | 权重 |
|------|---------|------|
| 同一文档 + 相邻条款 | 纵向 | 高 (≥0.8) |
| 同一专业 + 相关术语 | 横向 | 中 (0.5-0.8) |
| 跨专业 + 模糊关联 | 引用 | 低 (<0.5) |
