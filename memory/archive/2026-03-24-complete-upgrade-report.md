1. 每周查看 `memory/event-log.md` 了解系统事件
2. 每月检查 `memory/scheduler-state.json` 优化建议
3. 每季度审查架构文档，调整 Agent 策略

### 紧急干预
以下情况需要人工介入：
- 连续失败≥5 次的任务
- 系统资源持续告警（内存>95%）
- Gateway 无法自动重启
- 安全事件（未授权访问）
