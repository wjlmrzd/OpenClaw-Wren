### 1. 日常维护

#### daily-light-maintenance.ps1
- **用途**: 每日轻量维护
- **功能**:
  - 清理缓存
  - 重载配置
  - 检查 Agent 状态
  - 轻量重启
- **调用**: 每日维护员 (03:00)

#### log-rotate.ps1
- **用途**: 日志轮转
- **功能**:
  - 压缩旧日志
  - 删除过期日志

- **调用**: 日志清理员 (03:00)

#### check-disk.ps1
- **用途**: 磁盘检查
- **功能**:

  - 查找大文件
  - 清理临时文件

## 🔧 常用命令

### 手动执行脚本

```powershell
# 健康检查
.\scripts\health-check-simple.bat

# 自动修复
.\scripts\openclaw-auto-fix.bat

# 配置备份
.\scripts\backup-config.ps1

# 日志清理
.\scripts\log-rotate.ps1

# 安全审计
.\scripts\security-audit.bat
```

### 查看日志

```powershell
# 查看自动修复日志
type $env:USERPROFILE\.openclaw\logs\auto-fix.log

# 查看健康检查日志
type $env:USERPROFILE\.openclaw\logs\health-check.log
```
