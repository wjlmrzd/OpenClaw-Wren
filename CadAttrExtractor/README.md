# CadAttrExtractor

AutoCAD plugin for extracting block attributes, coordinate sorting, generating drawing index tables, and exporting to Excel/Word templates.

## 功能特性

### 核心功能
- **智能属性提取** - 通过 SelectionFilter + 正则表达式分组提取图块属性
- **四轴坐标排序** - 支持四种排序模式：
  - 从上到下，从左到右
  - 从左到右，从上到下
  - 从左到右，从下到上
  - 按选择顺序
- **DWG 表格生成** - 在 AutoCAD 中生成专业的图纸目录表格，支持自动分页
- **模板导出** - 导出到 Excel/Word 模板，支持占位符替换

### 用户界面
- **WPF 浮动面板** - 使用 PaletteSet 实现的浮动工具面板
- **MasterSkin2 主题** - 专业的深色主题支持
- **实时预览** - 提取数据实时预览，支持拖拽排序

### 其他特性
- **设置持久化** - 自动保存用户设置到 AppData
- **自动加载** - 通过 .bundle 包自动加载
- **日志记录** - 完整的运行日志

## 系统要求

- Windows 10/11 x64
- AutoCAD 2022 或更高版本
- Visual Studio 2022
- .NET Framework 4.8

## 安装说明

### 方法一：自动安装（推荐）
1. 复制整个 `CadTableExtractor.bundle` 文件夹到 AutoCAD 支持目录：
   ```
   %PROGRAMFILES%\Autodesk\AutoCAD 2022\Autodesk\AutoCAD 2022\Support\en-US\PalettePlugins\
   ```
   或
   ```
   %APPDATA%\Autodesk\AutoCAD 2022\enu\PalettePlugins\
   ```

2. 重启 AutoCAD，插件将自动加载

### 方法二：手动编译
1. 使用 Visual Studio 2022 打开 `CadTableExtractor.csproj`
2. 恢复 NuGet 包
3. 编译项目
4. 将输出 DLL 和 Bundle 内容部署到 AutoCAD 支持目录

## 使用指南

### 基本使用流程

#### 1. 打开面板
```
命令: _cte
或
命令: _cteShow
```

#### 2. 提取属性
1. 在"提取"标签页中设置过滤条件：
   - **图块名称** - 输入要提取的图块名称（支持通配符 *）
   - **属性标签** - 输入要提取的属性标签名
   - **图层** - 选择要提取的图层（可选）

2. 点击"提取"按钮
3. 在 AutoCAD 中框选要提取的图块
4. 系统将自动提取属性并显示在"预览"标签页

#### 3. 排序
在"预览"标签页中选择排序模式：
- **从上到下，从左到右** - 适合纵向图纸目录
- **从左到右，从上到下** - 适合横向图纸目录
- **从左到右，从下到上** - 适合逆向阅读顺序
- **按选择顺序** - 保持原始选择顺序

点击"应用排序"按钮。

#### 4. 生成表格
1. 切换到"表格"标签页
2. 设置表格参数：
   - 行高、列宽
   - 文字样式、高度
   - 表头文本
   - 分页设置
3. 点击"拾取起点"按钮，在绘图区域点击插入点
4. 点击"生成表格"按钮

#### 5. 导出
1. 切换到"导出"标签页
2. 选择模板：
   - Excel 模板（.xlsx）
   - Word 模板（.docx）
3. 设置属性标签映射：
   - 图名标签
   - 图号标签
   - 版本标签
   - 日期标签
4. 点击"导出 Excel"或"导出 Word"按钮

### 模板占位符

在 Excel/Word 模板中使用以下占位符：

| 占位符 | 说明 |
|--------|------|
| `{{DrawingTitle}}` | 图纸名称 |
| `{{DrawingIndex}}` | 图号/索引 |
| `{{DrawingTotal}}` | 总数 |
| `{{TotalCount}}` | 提取的项目总数 |
| `{{TotalPages}}` | 总页数 |
| `{{CurrentPage}}` | 当前页码 |
| `{{ExtractDate}}` | 提取日期 |
| `{{RowIndex}}` | 行序号 |

### 正则表达式模式

插件使用正则表达式从属性值中提取图纸信息：

```
模式1: (图名)(索引)    例: 总体设计(01)
模式2: 第X册/共Y册      例: 结构设计 第1册/共5册
模式3: 图名-索引       例: 电气图-001
模式4: 纯文本          其他情况
```

可在设置中自定义正则表达式模式。

## 设置说明

### 提取设置
- **容差** - 坐标排序时的容差值（默认 1.0）
- **包含匿名块** - 是否提取匿名图块
- **标题正则模式** - 自定义提取标题/索引的正则表达式

### 表格设置
- **行高** - 表格行高度（默认 8.0）
- **列宽** - 各列宽度设置
- **文字样式** - 表格文字样式名称
- **文字高度** - 表格文字高度
- **表头文本** - 表格标题文本
- **自动分页** - 启用自动分页
- **每页行数** - 每页最大行数

### 导出设置
- **Excel 模板路径** - Excel 模板文件位置
- **Word 模板路径** - Word 模板文件位置
- **上次导出目录** - 默认导出位置
- **属性标签映射** - 各字段对应的属性标签名

### 界面设置
- **自动显示面板** - 启动时自动显示面板
- **预览面板** - 显示预览功能
- **提取前确认** - 提取前显示确认对话框
- **记住排序模式** - 记住上次使用的排序模式

## 故障排除

### 插件不加载
1. 检查 Bundle 文件夹是否正确放置
2. 检查路径是否包含中文字符
3. 查看 AutoCAD 命令行错误信息

### 无法提取属性
1. 确认图块是否包含指定属性
2. 检查图块名称和属性标签是否正确
3. 确认图层是否正确

### 表格生成失败
1. 检查文字样式是否存在
2. 确认模型空间可用
3. 检查是否有足够的绘图空间

### 导出失败
1. 确认模板文件存在且格式正确
2. 检查模板中的占位符是否正确
3. 确认目标目录有写入权限
