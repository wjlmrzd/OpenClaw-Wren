## 核心组件

### 1. 主会话 (Main Session)
- **模型**: qwen3.5-plus
- **职责**: 实时响应用户请求，创建/管理子代理
- **位置**: `agents/main/`

### 2. 子代理系统 (Subagents)
- **创建方式**: 按需创建，用完即删
- **隔离**: 独立会话，独立上下文
- **用途**: 代码分析、数据处理、临时任务

### 3. Cron 任务集群
- **总数**: 18 个定时任务
- **分类**: 通知、监控、报告、备份、协调、保障
- **详情**: 见 [[02-cron-tasks]]

### 4. 任务协调员
- **频率**: 每天 21:00
- **职责**: 整合所有 Cron 任务输出，生成综合日报

---

## 运行模式

| 模式 | 触发条件 | 动作 |
|------|---------|------|
| 🟢 正常模式 | 默认 | 所有 Agent 正常运行 |
| 🟡 降载模式 | API>80% 或 资源>85% 或 失败≥3 | 暂停低优先级任务 |
| 🔴 安全模式 | 资源>95% 或 Gateway 故障 或 失败≥5 | 仅保留核心任务 |
