# PluginLoader 安装说明

## 📦 产品概述

**PluginLoader** 是一款专为 AutoCAD 和中望 CAD 设计的插件加载管理工具，提供集中化的插件管理解决方案。

### 主要功能

- ✅ 集中管理所有插件加载路径
- ✅ 主插件加载后自动加载配置的其他插件
- ✅ 交互式配置界面（DCL 对话框）
- ✅ 支持路径的增删改查操作
- ✅ 显示插件名称、版本、描述等详细信息

### 兼容平台

| 平台 | 版本支持 |
|------|----------|
| AutoCAD | 2014 - 2026 |
| 中望 CAD | 2024 - 2026 |

## 🔧 安装步骤

### 方法一：手动加载（推荐首次使用）

1. **复制文件**
   将整个 `CAD_PluginLoader` 文件夹复制到安全位置，建议：
   ```
   C:\CAD_Plugins\PluginLoader\
   ```

2. **加载主插件**
   在 AutoCAD/中望 CAD 命令行中输入：
   ```lisp
   (load "C:/CAD_Plugins/PluginLoader/PluginLoader.lsp")
   ```
   
   > **注意**：路径中的反斜杠需改为正斜杠，或使用双反斜杠

3. **验证加载**
   成功加载后会显示：
   ```
   PluginLoader v1.0.0 已加载。输入 PLM 打开管理界面。
   ```

### 方法二：自动加载（推荐日常使用）

1. **复制文件**
   将 `PluginLoader.lsp` 和 `PluginLoader.dcl` 复制到 CAD 支持路径：
   ```
   C:\Users\<用户名>\AppData\Roaming\Autodesk\AutoCAD 202X\RXX.X\chs\Support\
   ```

2. **添加启动加载**
   
   **方式 A：通过 APPLOAD 命令**
   - 输入 `APPLOAD` 命令
   - 在"启动组"中点击"内容"
   - 添加 `PluginLoader.lsp`
   
   **方式 B：修改 acaddoc.lsp**
   - 在支持路径中找到或创建 `acaddoc.lsp`
   - 添加以下代码：
   ```lisp
   (load "PluginLoader.lsp")
   ```

3. **配置支持路径**
   - 输入 `OPTIONS` 命令
   - 切换到"文件"选项卡
   - 展开"支持文件搜索路径"
   - 添加插件所在目录

## ⚙️ 配置说明

### 配置文件位置

配置文件默认保存在：
```
<CAD 启动目录>\PluginLoader\PluginLoader.cfg
```

### 配置文件格式

```ini
[Plugins]
; 格式：路径 | 名称 | 版本 | 描述 | 启用状态
C:\Plugins\Tool1.lsp|工具 1|1.0|功能描述|1
D:\Plugins\Tool2.vlx|工具 2|2.0|功能描述|0
```

### 字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| 路径 | 插件文件完整路径 | `C:\Plugins\Tool.lsp` |
| 名称 | 显示名称 | `绘图工具` |
| 版本 | 版本号 | `1.0` |
| 描述 | 功能描述 | `常用绘图命令` |
| 启用 | 1=启用，0=禁用 | `1` |

## 🚀 使用指南

### 命令列表

| 命令 | 功能 | 说明 |
|------|------|------|
| `PLM` | 打开管理界面 | 主命令，打开插件管理对话框 |
| `PLUGINLOADER` | 打开管理界面 | 完整命令名 |
| `PLR` | 重新加载插件 | 不打开界面，直接重新加载所有启用插件 |

### 管理界面操作

1. **打开界面**
   ```
   命令：PLM
   ```

2. **添加插件**
   - 点击"添加 (A)"按钮
   - 浏览选择插件文件 (.lsp/.fas/.vlx)
   - 填写插件信息
   - 点击确定

3. **启用/禁用插件**
   - 在列表中选择插件
   - 点击"启用/禁用 (T)"按钮
   - 或双击插件条目

4. **删除插件**
   - 在列表中选择插件
   - 点击"删除 (D)"按钮
   - 确认删除

5. **保存配置**
   - 点击"确定"按钮保存并重新加载
   - 点击"取消"放弃更改

## ⚠️ 注意事项

### 路径格式

- Windows 路径使用正斜杠或双反斜杠：
  ```lisp
  ; 正确
  "C:/Plugins/Tool.lsp"
  "C:\\Plugins\\Tool.lsp"
  
  ; 错误
  "C:\Plugins\Tool.lsp"  ; 单个反斜杠会被转义
  ```

### 文件权限

- 确保 CAD 对插件文件有读取权限
- 配置文件需要写入权限

### 兼容性

- AutoCAD 和中望 CAD 的 LISP 方言略有差异
- 如遇到问题，请检查命令行错误信息

### 性能建议

- 建议启用必要的插件，禁用不常用的
- 大量插件可能影响启动速度

## 🐛 故障排除

### 问题 1：无法加载对话框

**现象**：提示"无法加载对话框文件 PluginLoader.dcl"

**解决**：
1. 确认 `PluginLoader.dcl` 与 `PluginLoader.lsp` 在同一目录
2. 或将 DCL 文件所在目录添加到 CAD 支持路径

### 问题 2：插件不自动加载

**现象**：启动 CAD 后插件未自动加载

**解决**：
1. 检查配置文件中插件状态是否为"1"
2. 检查插件文件路径是否正确
3. 使用 `PLR` 命令手动重新加载

### 问题 3：配置文件无法保存

**现象**：修改配置后无法保存

**解决**：
1. 检查配置文件所在目录的写入权限
2. 确认文件未被其他程序占用
